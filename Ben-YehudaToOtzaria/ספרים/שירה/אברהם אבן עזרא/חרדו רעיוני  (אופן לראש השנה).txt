<h1>חרדו רעיוני (אופן לראש השנה)</h1>
אברהם אבן עזרא
<blockquote><p>חָרְדוּ רַעְיוֹנַי, / כִּי אֱלֹהֵי מִשְׁפָּט / שׁוֹכֵן עַד וְקָדוֹשׁ, /מְדַבֵּר בִּצְדָקָה.</p></blockquote>
<blockquote><p>בְּאָהֳבוֹ עַם הֲמוֹנַי, / שָׂם לוֹ חֹק וּמִשְׁפָּט – / וְדָת יוֹם זֶהקָדוֹשׁ / תִּהְיֶה לוֹ צְדָקָה,</p></blockquote>
<blockquote><p>וְיִגְבַּהּ יְיָ / צְבָאוֹת בַּמִּשְׁפָּט / וְהָאֵל הַקָּדוֹשׁ /נִקְדָּשׁ בִּצְדָקָה.</p><p>אַבִּירֵי בְרִיאוֹת / אֶרֶץ, הָחִישׁוּ / לְרַצּוֹת בּוֹרַאֲכֶם /וְחַסְדּוֹ יַאֲמִצְכֶם,</p></blockquote>
<blockquote><p>וּבִנְפָשׁוֹת צְמֵאוֹת / לוֹ שִׁיר הַגִּישׁוּ, / הוּא יְרַוֶּה צְמָאֲכֶם/ וְדִשְׁנוֹ יַחֲלִיצְכֶם –</p></blockquote>
<blockquote><p>אֶת יְיָ צְבָאוֹת – / אֹתוֹ תַקְדִּישׁוּ, / וְהוּא מוֹרַאֲכֶם / וְהוּאמַעֲרִצְכֶם.</p><p>בְּמִבְטָא שְׂפָתַיִם / לְךָ אֶעֱרֹךְ רְנָנָי / וְעִנְיְנֵי תְהִלָּתְךָ/ עֲלֵי לִבִּי חֲרוּשִׁים;</p></blockquote>
<blockquote><p>גָּדוֹל בְּמַחֲנַיִם / שִׁמְךָ, מְעוֹן מְעוֹנָי: / בְּרוֹם –בַּהֲמֻלָּתְךָ, / לְמַטָּה – בַּנְּפָשִׁים,</p></blockquote>
<blockquote><p>וְיוֹדוּ שָׁמַיִם / פִּלְאֲךָ, יְיָ, / אַף אֱמוּנָתְךָ / בִּקְהַלקְדֹשִׁים.</p><p>רָאֲךָ מִבְּשָׂרִי / לֵב, שַׂמְתּוֹ לְצוֹפֶה, / שְׂעִפַּי – הוֹשִׁיבָם /בִּמְקוֹם הַקֹּדֶשׁ.</p></blockquote>
<blockquote><p>מְאֹד אֶשְׂמַח, / בְּזָכְרִי / יוֹם תֵּת חֹק יָפֶה / לְבָנִים, בְּבוֹאמְחוֹבְבָם / מֵרִבְבוֹת קֹדֶשׁ –</p><p>רֶכֶב אֱלֹהִים רִ-/ בֹּתַיִם אַלְפֵי / שִׁנְאָן, יְיָ בָם / סִינַיבַּקֹּדֶשׁ.</p><p>הַכֹּל מַגִּידִים / גְּבוּרָתוֹ וְגָדְלוֹ / בְּעוֹלָמִים שְׁנַיִם /יְהַלֲלוּהוּ יָחַד:</p></blockquote>
<blockquote><p>מִשְׁתַּחֲוִים וּמוֹדִים / מְתִים חוֹסִים בְּצִלּוֹ, / וּצְבָאהַשָּׁמַיִם / יְשָׁרְתוּהוּ בְּפָחַד –</p></blockquote>
<blockquote><p>שְׂרָפִים עֹמְדִים / מִמַּעַל לוֹ, / שֵׁשׁ כְּנָפַיִם / שֵׁשׁ כְּנָפַיִםלְאֶחָד.</p><p>מְבַקְּשִׁים אַיֵּה זֶה / מַחְנֶךָ, קָדוֹשׁ, / בְּנֵי אֱלֹהִים, כִּיאוֹת / כְּבוֹדְךָ בָּם וְהוֹדוֹ;</p></blockquote>
<blockquote><p>רָאָהוּ בְמַחֲזֶה / אִישׁ אֱלֹהִים קָדוֹשׁ / עַל כִּסֵּא פְלָאוֹת /וּסְבִיבָיו גְּדוּדוֹ –</p></blockquote>
<blockquote><p>וְקָרָא זֶה אֶל זֶה / וְאָמַר: קָדוֹשׁ, קָדוֹשׁ, קָדוֹשׁ / יְיָצְבָאוֹת, / מְלֹא כָל הָאָרֶץ כְּבוֹדוֹ!</p></blockquote>
