<blockquote><p>דּוֹדֵי לֵב<a class="footnote" href="#fn:8" id="fnref:8" title="see footnote"><sup>8</sup></a>, אֲשֶׁר כָּלפִּרְיָן // לִי טֹבוּ,</p></blockquote>
<blockquote><p>לָשׂוּם עַל כְּאֵבִי צָרְיָן // לֹאאָבוּ<a class="footnote" href="#fn:9" id="fnref:9" title="see footnote"><sup>9</sup></a>. עיני</p></blockquote>
<blockquote><p>דִּבַּרְתִּי<a class="footnote" href="#fn:10" id="fnref:10" title="see footnote"><sup>10</sup></a> בְעָמְדִילִפְנֵי // מַשְׂטִינָהּ<a class="footnote" href="#fn:11" id="fnref:11" title="see footnote"><sup>11</sup></a></p></blockquote>

<blockquote><p>לֹא תָחֹן לְךָ<a class="footnote" href="#fn:12" id="fnref:12" title="see footnote"><sup>12</sup></a> וּבְעֵינֵי// כֹל חִנָּהּ</p></blockquote>
<blockquote><p>עֵת תַּהְרֹס וְגַם עֵת תִּבְנֶה<a class="footnote" href="#fn:13" id="fnref:13" title="see footnote"><sup>13</sup></a> // בִּנְיָנָהּ</p></blockquote>
<blockquote><p>אַל נָא<a class="footnote" href="#fn:14" id="fnref:14" title="see footnote"><sup>14</sup></a> בָהּ עֲלֵי כָלעִנְיָן // תָּרִיבוּ,</p></blockquote>
<blockquote><p>כִּי אִם מַהֲרָסָהּ בִּנְיָן<a class="footnote" href="#fn:15" id="fnref:15" title="see footnote"><sup>15</sup></a> // תַּחְשֹׁבוּ. // עיני</p></blockquote>
<blockquote><p>לִבִּי יֶהֱמֶה כַּתַּנִּים<a class="footnote" href="#fn:16" id="fnref:16" title="see footnote"><sup>16</sup></a> // יִצְוָחוּ<a class="footnote" href="#fn:17" id="fnref:17" title="see footnote"><sup>17</sup></a></p></blockquote>

<blockquote><p>אֶת סוֹדִי<a class="footnote" href="#fn:21" id="fnref:21" title="see footnote"><sup>21</sup></a> כְּמֵימֵימַעְיָן // שָׁאָבוּ</p></blockquote>
<blockquote><p>וּלְבָבִי בְּרֶגֶשׁ חֶלְיָן // לִבֵּבוּ.<a class="footnote" href="#fn:22" id="fnref:22" title="see footnote"><sup>22</sup></a>// עיני</p></blockquote>

<blockquote><p>כִּי שִׁלְטוֹן צְבִי כִּמְאוֹרִים // כֵּן יִזְרָח</p></blockquote>
<blockquote><p>וּבְהֶמְיַת צְבִי אֶת מוֹרִים // הוּאיַבְרַח<a class="footnote" href="#fn:23" id="fnref:23" title="see footnote"><sup>23</sup></a></p></blockquote>
