<h1>[לא כמשפטי]</h1>
שמואל הנגיד
<blockquote><p>— ∪ — — —∪ — — — — — // — ∪ — — —∪ — — — — —</p></blockquote>
<blockquote><p>לֹא כְמִשְׁפָּטִי תְדִינוּן עָלֵימוֹ // חַפְּשׂוּ הֵיטֵבוְדִינוּם כִּשְׁלֹמֹה<a class="footnote" href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a></p></blockquote>
<blockquote><p>מַהֲרִי שׁוּבִי, תְּנוּמַת עֵינֵימוֹ, // וַאֲנִי מֵאֵיןתְּנוּמָה אֶבְכֵּמוֹ</p></blockquote>
<blockquote><p>הֵן לְבָבִי סָר<a class="footnote" href="#fn:2" id="fnref:2" title="see footnote"><sup>2</sup></a> וְנִמְהַר<a class="footnote" href="#fn:3" id="fnref:3" title="see footnote"><sup>3</sup></a> אַחְרֵימוֹ, // כִּימְנַהֵג בָּם, לְהַכְעִיס, מִהְרָמוֹ<a class="footnote" href="#fn:4" id="fnref:4" title="see footnote"><sup>4</sup></a>.</p></blockquote>
<blockquote><p>שַׁאֲלוּ נָא אֶת שְׁלוֹמִי<a class="footnote" href="#fn:5" id="fnref:5" title="see footnote"><sup>5</sup></a>,כִּי כָל דּוֹד // אֶל יְדִידוֹ סָר וְשׁוֹאֵל בִּשְׁלוֹמוֹ</p></blockquote>
<blockquote><p>כָּהֲתָה עֵין אוֹהֲבָם<a class="footnote" href="#fn:6" id="fnref:6" title="see footnote"><sup>6</sup></a> כִּיאֵינֵימוֹ, // אַשֲׁרֵי עַיִן אֲשֶׁר עוֹד תֶּחְזֵמוֹ!</p></blockquote>
<ol><li id="fn:1"><p>אל תדונו, האנשים, את אוהבי כמשפטי הנמהר שרחצתיעליכם בשעת צערי, אלא דרשו וחקרו היטב ודינום בחכמה כשלמה. <a class="reversefootnote" href="#fnref:1" title="return to body"> ↩</a></p></li><li id="fn:2"><p><strong>סר</strong> – יצא. <a class="reversefootnote" href="#fnref:2" title="return to body"> ↩</a></p></li><li id="fn:3"><p><strong>ונמהר</strong> – ומהר. <a class="reversefootnote" href="#fnref:3" title="return to body"> ↩</a></p></li><li id="fn:4"><p><strong>כי מנהג בם להכעיס מהרמו</strong> – כי המוביל אותםבדרך מהר להרחיקם ממני להכעיסני. <a class="reversefootnote" href="#fnref:4" title="return to body"> ↩</a></p></li><li id="fn:5"><p><strong>שאלו נא את שלומי</strong>־ אתם ידידי שהלכתם ממני,לפחות שאלו לשלומי. <a class="reversefootnote" href="#fnref:5" title="return to body"> ↩</a></p></li><li id="fn:6"><p><strong>אוהבם</strong> – המשורר. <a class="reversefootnote" href="#fnref:6" title="return to body"> ↩</a></p></li></ol>
