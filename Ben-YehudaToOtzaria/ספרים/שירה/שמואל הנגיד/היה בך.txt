<h1>[היה בך]</h1>
שמואל הנגיד
<p>— — ∪ — — — ∪— — — ∪ — // — — ∪ — — — ∪ — — — ∪ —</p>
<blockquote><p>הָיָה בְךָ שִׂבְרִי<a class="footnote" href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a> — וְעַליָדָךְ<a class="footnote" href="#fn:2" id="fnref:2" title="see footnote"><sup>2</sup></a> כְּבָר // רָאוּ מְשַׂנְאַי אֶתאֲשֶׁר בִּי שִׂבְּרוּ<a class="footnote" href="#fn:3" id="fnref:3" title="see footnote"><sup>3</sup></a>.</p></blockquote>
<blockquote><p>הָלְכוּ<a class="footnote" href="#fn:4" id="fnref:4" title="see footnote"><sup>4</sup></a> עֲלֵי דֶרֶךְנְתִיבָה<a class="footnote" href="#fn:5" id="fnref:5" title="see footnote"><sup>5</sup></a> לַעֲשׂוֹת // חֶפְצָם אֲשֶׁרחָפְצוּ — וְחֶפְצִי אֵחֲרוּ<a class="footnote" href="#fn:6" id="fnref:6" title="see footnote"><sup>6</sup></a></p></blockquote>
<blockquote><p>אִלּוּ תְשֻׁלַּח<a class="footnote" href="#fn:7" id="fnref:7" title="see footnote"><sup>7</sup></a> אֵשׁנְדוּדִים בָּם<a class="footnote" href="#fn:8" id="fnref:8" title="see footnote"><sup>8</sup></a> כְּמוֹ // בִישֻׁלְּחָה<a class="footnote" href="#fn:9" id="fnref:9" title="see footnote"><sup>9</sup></a> — שָׁעוּ<a class="footnote" href="#fn:10" id="fnref:10" title="see footnote"><sup>10</sup></a> וְלָשׁוּב מִהֲרוּ.</p></blockquote>
<blockquote><p>לָמָּה שְׁכַחְתֶּם אֶת בְּרִיתִי? הַעֲלוּ // עַל לֵביְדִידוּתִי, לְמַעַן תִּזְכְּרוּ!</p><br/></blockquote>
